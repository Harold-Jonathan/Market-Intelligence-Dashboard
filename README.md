# 🚀 Market Intelligence & RPA Pipeline

Este projeto apresenta um ecossistema completo de automação e análise de dados. Ele integra um robô **RPA (UiPath)** com uma camada de **Data Analytics (Python/Colab)** para transformar buscas brutas no Mercado Livre em decisões estratégicas de negócio.

## 🛠️ O Ciclo de Vida do Dado
1. **Coleta (RPA):** O robô navega no Mercado Livre, extrai metadados de acessibilidade (ARIA-labels) e normaliza preços usando Regex.
2. **Processamento (Python/Pandas):** Limpeza de outliers, tratamento de moedas e cálculo de KPIs de custo-benefício.
3. **Visualização (BI):** Geração de um Dashboard Executivo interativo em HTML/CSS com links diretos para compra.
